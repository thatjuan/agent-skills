## Core Quality Tests

Six tests applied to each logo concept before finalization.

### 1. Napkin Test

**Question:** Can someone draw this logo from memory on a napkin after seeing it for 5 seconds?

**Pass criteria:**
- Essential form is reproducible by a non-designer
- No more than 3-4 distinct elements
- Core shape is simple enough to describe in one sentence

**Fail signals:**
- Requires precise curves or proportions to be recognizable
- Has more than 5 distinct elements
- Details are load-bearing (remove one and meaning breaks)

### 2. Thumbnail Test

**Question:** Is the logo recognizable at 16×16 pixels (favicon size)?

**Pass criteria:**
- Silhouette alone is identifiable at tiny sizes
- Color is not required for recognition
- No fine details that collapse into noise

**Assessment method:** Mentally reduce the logo to a 16px square. Elements that disappear at this scale are liabilities.

### 3. Silhouette Test

**Question:** Is the logo identifiable as a single-color silhouette?

**Pass criteria:**
- The outline alone carries the brand identity
- Internal color variations are supplementary, not essential
- Shape is distinctive enough to own

### 4. Black & White Test

**Question:** Does the logo work without color?

**Pass criteria:**
- Contrast ratios are sufficient in monochrome
- No information is lost when color is removed
- The mark feels intentional in B&W, not like a degraded version

### 5. Swap Test

**Question:** If this logo were placed on a competitor's product, would it look wrong?

**Pass criteria:**
- The mark feels specific to this brand
- Visual language doesn't match competitor conventions too closely
- There is at least one element that is distinctly ownable

**Fail signals:**
- Logo could belong to any company in the category
- Visual style matches the market leader too closely
- Nothing about the mark is specific to this brand's story

### 6. Context Test

**Question:** Does the logo hold its own when placed next to competitor logos at real-world scale?

**Assessment method:**
- Place the concept alongside 5+ competitor logos
- View at actual application sizes (app icon grid, search results, storefront)
- Check: does it stand out or blend in? Does it look like it belongs in the category while being distinct?

## Memorability Drivers

Three factors that determine whether a logo is remembered.

### Simplicity

The fewer elements, the stronger the memory trace. The brain encodes simple shapes more reliably than complex illustrations.

**Benchmark:** The world's most recognized logos (Apple, Nike, McDonald's, Mercedes) are all reducible to one or two basic shapes.

### Distinctiveness

A logo's memorability is proportional to how different it is from everything else in the viewer's visual environment.

**Measurement:** Count the number of visual features that distinguish the mark from competitors. Fewer than two = insufficient differentiation.

### Relevance

The logo connects to something the viewer already cares about. This connection can be literal (pictorial) or emotional (through color, shape, and cultural associations).

**Check:** Does the mark trigger an association with the brand's core value proposition?

## Scalability Assessment

### Size Range Requirements

| Context | Size | Priority |
|---------|------|----------|
| Favicon | 16×16 px | Recognition |
| App icon | 48-512 px | Clarity |
| Social avatar | 80-200 px | Personality |
| Business card | ~25mm | Detail |
| Website header | 120-300 px wide | Legibility |
| Signage | 1-10 meters | Impact |
| Billboard | 5+ meters | Distance legibility |

### Design-First-Small Principle

Designing at the smallest required size first, then scaling up, prevents detail-dependency. Elements that work at 16px work at any size. The reverse is not true.

### Responsive Variants

Complex logos may need simplified variants for small sizes:

| Size Range | Variant |
|------------|---------|
| < 32px | Symbol only, simplified |
| 32-120px | Symbol only, full detail |
| 120-300px | Combination mark (symbol + wordmark) |
| > 300px | Full lockup with tagline (if applicable) |

## Market Fit Evaluation

### Audience Resonance Test

**Question:** "If this brand were a person, who would it be?"

If the answer aligns with the target audience's aspirations (who they want to become, not who they are now), the concept has market fit.

### Category Recognition

The logo's visual language communicates what category the brand operates in. At least one visual element (color, shape, or typography style) aligns with category conventions to establish immediate context.

### Differentiation Sufficiency

**Minimum threshold:** At least 2 of 5 visual levers (color, shape, typography, abstraction, negative space) differ from the category norm.

**Maximum threshold:** No more than 4 of 5 should differ — complete visual divergence risks appearing to be from a different category entirely.

## Iteration Decision Framework

### When to Refine (Incremental)

- The concept direction is right but execution needs polish
- User says "I like X but..." with a specific adjustment
- One element is strong, another needs work
- Proportions or weight feel slightly off

**Action:** 3-5 variations adjusting the specified elements.

### When to Combine (Synthesis)

- User points to elements from different concepts: "A1's icon with B2's typography"
- Two concepts each have one excellent element
- The hybrid possibility is clearly stronger than either parent

**Action:** Produce 2-3 combinations, maintaining the structural logic of each source element.

### When to Redirect (Reset)

- User rejects all 9 concepts
- Feedback suggests the strategic positioning was wrong ("this doesn't feel like us")
- Discovery reveals new information that changes the brief

**Action:** Return to Phase 1. Update the Only Statement, reassess the archetype, reposition on personality spectrums.

### When to Expand (Deepen)

- User gravitates toward one direction: "I want more like C-row"
- The direction is right but the specific concepts don't click
- Additional logo types (lettermark, emblem) might unlock the concept

**Action:** Generate 3-6 new concepts within the selected direction, exploring different logo types and visual metaphors.

### Convergence Signal

A concept is final when it passes all 6 core quality tests AND the user confirms it represents their brand. The rationale document (why this mark works) accompanies the final selection.
