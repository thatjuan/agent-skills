# Agent Skills

A collection of AI agent skills for Claude Code and other compatible agents.

## Skills

| Skill | Description |
|-------|-------------|
| [team-executor](./team-executor/) | Multi-agent orchestration that transforms braindumps into executed results through expert planning and autonomous execution |

## Installation

```bash
# Install a specific skill
npx skills add thatjuan/agent-skills --skill team-executor

# Install all skills
npx skills add thatjuan/agent-skills --all
```
