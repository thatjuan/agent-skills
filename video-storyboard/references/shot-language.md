# Shot Language

Precise vocabulary for describing shots, camera, lighting, and transitions in a storyboard. Using exact terms makes the text storyboard filmable — a director, DP, and editor can read it and execute without ambiguity.

## Shot Sizes

| Term | Abbrev | What it Frames |
|------|--------|----------------|
| Extreme Wide / Establishing | EWS / ELS | Subject is small or absent — sets place and scale |
| Wide / Long Shot | WS / LS | Full subject head-to-toe with environment |
| Full Shot | FS | Subject head-to-toe, environment minimal |
| Medium Wide / Cowboy | MWS | Subject from mid-thigh up |
| Medium | MS | Subject from waist up |
| Medium Close-Up | MCU | Chest up |
| Close-Up | CU | Head/face fills frame |
| Extreme Close-Up | ECU | Eye, mouth, single feature, or product detail |
| Insert | INS | Hand-action, product detail, screen replacement |
| Two-Shot / Three-Shot | 2S / 3S | Two or three subjects framed together |
| OTS (Over-the-Shoulder) | OTS | Frame partially occluded by foreground subject's shoulder |
| Point-of-View | POV | Camera as character's eyes |

## Camera Angles

| Term | Effect |
|------|--------|
| **Eye-level** | Neutral, equal — default for dialogue and naturalism |
| **High angle** | Looking down on subject — diminishes, isolates, makes vulnerable |
| **Low angle** | Looking up at subject — empowers, heroizes, intimidates |
| **Bird's-eye / Top-down / Overhead** | Directly above — graphic, organizational, fate-of-the-scene |
| **Worm's-eye** | Directly below — extreme heroization or surreal |
| **Dutch tilt / Canted** | Camera rolled — unease, instability, dream logic |
| **Hip / Low-mounted** | Below eye — kinetic, embodied |

## Lens Choice

Lens is shorthand for *feel*. Specify focal length when it matters.

| Lens | Focal Length (35mm equiv.) | Feel |
|------|---------------------------|------|
| Ultra-wide | 14–20mm | Distorted, environmental, immersive |
| Wide | 24–28mm | Spacious, journalistic, slightly distorted at edges |
| "Normal" | 35–50mm | Human-eye-like, intimate, contemporary |
| Portrait | 85mm | Flattering compression, romantic, premium |
| Telephoto | 100–200mm | Compressed, voyeuristic, cinematic isolation |
| Macro | n/a | Extreme magnification — texture, product detail |
| Anamorphic | n/a | Cinematic widescreen, oval bokeh, horizontal lens flare |
| Probe / Snorkel | n/a | Extreme proximity inserts (food, product hero) |

Specify aperture for depth of field when material: *"shot wide open at T1.4 — eyes razor-sharp, ear out of focus."*

## Camera Movement

| Term | Description |
|------|-------------|
| **Static / Locked off** | Camera does not move |
| **Pan** | Rotates horizontally on its axis |
| **Tilt** | Rotates vertically on its axis |
| **Roll** | Rotates along the lens axis |
| **Dolly in / out** | Camera body moves toward or away on the floor |
| **Track / Truck** | Lateral camera movement, parallel to subject |
| **Push-in / Pull-out** | Slow dolly in or out for emotional emphasis |
| **Crane / Jib** | Vertical sweep on a boom arm |
| **Handheld** | Operator-held, organic motion |
| **Steadicam / Gimbal** | Stabilized but moving — fluid, immersive |
| **Drone / Aerial** | Airborne perspective |
| **Whip pan** | Fast pan, often used as a transition |
| **Snap zoom** | Sudden zoom — comedic or aggressive |
| **Dolly zoom (Vertigo)** | Dolly + opposing zoom — vertiginous warping |
| **Crash zoom** | Aggressive sudden zoom in |
| **Rack focus** | Focus shifts between two subjects in frame |
| **Arc** | Camera circles subject |
| **Reveal move** | Camera motion uncovers something previously hidden |
| **Tabletop / Bolt rig** | Programmed precision moves over product (food, beauty, tech) |
| **Time slice / Bullet time** | Multi-camera frozen-moment sweep |

## Composition & Framing

- **Rule of thirds** — subject on intersecting third
- **Center punch** — subject dead center, often symmetrical
- **Symmetrical / one-point** — Wes Anderson signature
- **Negative space** — large empty regions amplifying the subject
- **Foreground / Midground / Background** — layered depth
- **Leading lines** — graphic vectors directing eye
- **Frame within a frame** — doorways, windows, mirrors as internal framing
- **Headroom / Lookroom / Noseroom** — space above head or in direction of gaze
- **Eyeline** — line of sight, must match across cuts in dialogue

## Lighting Vocabulary

| Term | Description |
|------|-------------|
| **Key light** | Primary directional source |
| **Fill** | Softens shadows opposite the key |
| **Backlight / Rim** | Behind subject — separation halo |
| **Practical** | Light visible in frame (lamp, neon, screen) |
| **Motivated** | Apparent source matches the world (window, fire) |
| **High-key** | Bright, low-contrast — comedy, beauty, premium |
| **Low-key** | Dark, high-contrast — drama, noir, cinematic |
| **Naturalistic** | Window light, sun, available — documentary feel |
| **Hard light** | Sharp shadows — drama, edge, harshness |
| **Soft light** | Diffused — flattery, warmth, beauty |
| **Top light** | From above — moody, theatrical |
| **Underlight** | Below subject — sinister, surreal |
| **Side light / Rembrandt / Split** | Half-lit face — psychological |
| **Bounce** | Light bounced off a card or surface |
| **Day-for-night / Night-for-day** | Faked time of day in grade |
| **Magic hour / Golden hour / Blue hour** | Specific times of day with signature looks |
| **Color temperature** | Warm (3200K tungsten) vs. cool (5600K daylight) |
| **Gel / Color cast** | Colored filter on the source |

## Color & Look

- **Palette** — describe in 2–3 dominant hues + accent ("dusty terracotta + blue shadow + buttery sodium streetlight accent")
- **Saturation** — desaturated, muted, naturalistic, hyper-saturated
- **Contrast** — flat, lifted, crushed-blacks
- **Grade reference** — *"think* The Bear *kitchen" / "Apple Vision Pro launch palette" / "early Spike Jonze MTV"*
- **Film stock reference** — Kodak Portra warmth, Fuji 400H pastel cool, Ektachrome saturated daylight, 16mm grain, super-8 home-movie
- **Texture** — clean digital, subtle grain, heavy 16mm grain, halation, gate weave, video artefacts

## Time / Speed

- **Real-time** — 24fps default
- **Slow-motion** — 48fps (2x), 96fps (4x), 240fps (10x — extreme)
- **Speed ramp** — variable speed within a shot, often slow→fast or fast→slow
- **Time-lapse / Hyperlapse** — accelerated time
- **Stop-motion** — frame-by-frame animation
- **Freeze frame** — held still moment
- **Reverse** — playback in reverse

## Transitions

| Type | Effect |
|------|--------|
| **Hard cut** | Default — one frame to the next |
| **Match cut** | Visual match across cut (shape, motion, color) |
| **Graphic match** | Two visually similar compositions cut together |
| **J-cut** | Audio of next scene starts before its picture |
| **L-cut** | Audio of previous scene continues into next picture |
| **Smash cut** | Aggressive cut against the grain — for comedy or shock |
| **Whip pan transition** | Camera whip ends one shot, begins next |
| **Cross-dissolve** | Soft fade between two shots — passage of time, dream |
| **Fade in / Fade out** | To/from black or white |
| **Wipe** | One image pushes another off frame |
| **Iris in/out** | Circular open or close |
| **Morph** | Shapeshifting transition |
| **Speed-blur transition** | Motion-blur masked cut |
| **Object pass-by** | Foreground object briefly fills frame, hiding the cut |
| **Whip zoom** | Rapid zoom carries the cut |
| **Glitch / Datamosh** | Digital artifact transition |

## Audio Terminology

- **Diegetic** — sound from within the world (footsteps, dialogue, in-world music)
- **Non-diegetic** — score, V/O, sound design from outside the world
- **Sync sound** — recorded with picture (dialogue)
- **ADR / VO booth** — recorded separately
- **Foley** — performed sound effects (footsteps, fabric, prop handling)
- **Sound design** — designed effects (whooshes, drones, impacts, risers)
- **Mnemonic / Sonic logo** — short signature sound ending the spot
- **Needle drop** — licensed pre-existing track
- **Score / Bespoke** — original music written for the spot
- **Stinger** — short musical hit punctuating a moment
- **Drop / Build** — energy structure of a track
- **Riser / Whoosh / Impact** — design transitions in audio
- **Ambient bed / Room tone** — sustained environmental sound underlying dialogue

## On-Screen Graphics

- **Super / Lower-third** — text overlaid on picture
- **Kinetic type** — animated text driving the spot
- **Title card** — full-frame text, often between scenes
- **Lower-third bug** — small persistent logo or label
- **End card / End frame** — final logo + tagline + CTA
- **Legal super** — small-print disclosure, usually bottom of frame, last 2–3s
- **Chyron** — broadcast lower-third
- **Pack shot** — product shot, usually end-of-spot
- **Product overlay** — graphic callout pointing to product feature
- **UI replacement / Screen comp** — composited phone or laptop screen
